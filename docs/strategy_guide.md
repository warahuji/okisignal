# SMC_FVG シグナル配信EA — 運営者ガイド

## ロジック概要

**一言で言うと：** 相場の「構造転換」を検知し、価格の「隙間」に戻ってきたところでエントリーする

---

## 3ステップのロジック

### ステップ1: Swing Point（振り子の頂点）を見つける

```
        Swing High
           /\
          /  \
         /    \
        /      \
       /        \        /\
      /          \      /  \
                  \    /
                   \  /
                    \/
                 Swing Low
```

- **Swing High** = 周りの足より高い「山の頂点」
- **Swing Low** = 周りの足より低い「谷の底」
- SwingLen本の足で確認（ペアごとに3〜8本で設定）

### ステップ2: Structure Break（構造転換 = BOS）

```
  前のSwing High -------- この水平線を終値が超えたら「上昇転換」
           /\
          /  \
         /    \      /\
        /      \    /  \
                \  /    \
                 \/      \  ← 終値がここを下抜けたら「下降転換」
           前のSwing Low --------
```

- **Bullish BOS（上昇転換）**: 終値が直近のSwing Highを**上抜け**
- **Bearish BOS（下降転換）**: 終値が直近のSwing Lowを**下抜け**
- これで「相場の方向」が決まる

### ステップ3: FVG（Fair Value Gap = 価格の隙間）でエントリー

```
  BOS発生後、勢いよく動いた3本の足:

  足1 |████|  ← high = 149.50
  足2 |████████████|  ← 大きなインパルス（勢いの足）
  足3      |████|  ← low = 149.70

  → 足1のhigh(149.50) と 足3のlow(149.70) の間に「隙間」がある
  → この隙間 = FVG（フェアバリューギャップ）
  → 価格がこの隙間に戻ってきたらエントリー！
```

**なぜ機能するか：**
- 大口（銀行・機関投資家）が急いで注文を入れた痕跡
- 隙間に戻ると、残りの注文が執行されて再び同じ方向に動きやすい

## フィルター（無駄なトレードを減らす仕組み）

### セッションフィルター（時間帯制限）
- 通貨ペアごとに「トレードする時間帯」を制限
- 例: EURUSDは14-19時GMT（NYセッション）のみ
- 流動性が低い時間帯のダマしを回避

### ADRフィルター（1日の値幅制限）
- ADR = Average Daily Range（過去20日の平均日足値幅）
- 当日の消費率が設定値を超えたらエントリーしない
- 「今日はもう十分動いた」→ 反転リスクが高いのでスキップ

## トレード管理（エントリー後）

```
  エントリー  TP1（半分決済）    TP2（全決済）
     |           |                |
     ●===========●================●
     |
     |--- SL（損切り）

  1. エントリー → SLとTP1を設定して発注
  2. TP1に到達 → ロットの50%を利確
  3. SLをエントリー価格に移動（= ブレイクイーブン、損失ゼロ保証）
  4. 残り50%でTP2を狙う
```

- **SL/TP はATR（平均的な値動き幅）ベースで自動計算**
- ペアごとに倍率が違う（最適化済み）
- ロットサイズは残高の1%リスクで自動計算（複利）

## 運用上の注意

1. **EA再起動時**: 過去100本の足を自動スキャンして即トレード可能（WarmUp機能）
2. **1チャートで全ペア**: どのチャートに貼ってもOK（OnTimerで全ペア巡回）
3. **VPS必須**: 24時間稼働が前提。PC電源OFFでEA停止
4. **デモ口座推奨**: まずデモで数日〜1週間フォワードテスト
5. **KIWAMI口座**: スプレッドが狭い口座でないと成績が大幅悪化
